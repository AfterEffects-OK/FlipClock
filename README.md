# レトロフリップクロック (Retro Flip Clock)

ブラウザで動作する、レトロなデザインのフリップ式デジタル時計です。
時刻や日付がめくれるように切り替わるアニメーションが特徴です。

## ✨ 主な機能

*   **時刻表示**: 時・分・秒をフリップアニメーションで表示します。
*   **日付表示**: 年・月・日・曜日を表示します。
*   **レスポンシブデザイン**: PCでもスマートフォンでも見やすいレイアウトに自動で調整されます。
*   **フルスクリーン表示**: 時計を画面全体に表示できます。
*   **多彩なカスタマイズオプション**:
    *   **時刻形式**: 24時間表示と12時間表示を切り替え可能です。
    *   **曜日言語**: 日本語と英語を切り替え可能です。
    *   **テーマ**:
        *   **黒**: デフォルトのダークテーマです。
        *   **白**: 明るいライトテーマです。
        *   **透明**: 背景が透過し、デスクトップアプリのように利用できます。
        *   **カスタム**: カラーピッカーでお好みの背景色を設定できます。

## 🚀 使い方

1.  `index.html` ファイルをウェブブラウザで開くだけで利用できます。
2.  画面右上の歯車 <i class="fas fa-cog"></i> アイコンをクリックすると、設定パネルが開きます。
3.  設定パネルから、時刻形式、曜日の言語、テーマを自由に変更できます。
4.  画面右上の拡大 <i class="fas fa-expand"></i> アイコンをクリックすると、フルスクリーン表示に切り替わります。
